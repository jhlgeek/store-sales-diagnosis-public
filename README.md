# 外卖平台门店销售诊断（公开版 / 匿名化）

> 目标：用**真实业务场景**（线上外卖平台）做一个“可复现”的数据分析项目作品集。  
> 数据已做匿名化处理（门店/城市/敏感字段均已替换），仅用于展示分析方法与思路。

## 1. 项目背景与问题定义

这是多家门店在同一外卖平台 **线上渠道** 的订单与商品销售数据（不包含线下销售）。  
本项目围绕一个典型经营问题展开：

- **问题 1：** 本月 GMV（销售额）波动与峰值来自哪里？  
- **问题 2：** 若出现 GMV 下滑，主要是哪些门店贡献了下滑？  
- **问题 3：** 在“价格相对稳定”的前提下，GMV 变化更可能由哪些经营因子驱动（订单量、SKU 活跃度、订单结构等）？

> GMV（Gross Merchandise Value）在这里指：线上渠道的商品销售额。

## 3. 分析方法与核心指标

### 3.1 指标拆解（主线）
- **GMV = Orders × AOV**
  - Orders：订单量
  - AOV：客单价（每单销售额）

在“商品标价整体稳定”的业务背景下，GMV 波动往往更多来自 **订单量变化** 或 **订单结构变化**（例如活动、套餐、某类下单入口）。

### 3.2 关键辅助指标
- **Active SKU**：当日活跃 SKU 数（用于观察商品供给/活动上新对订单的影响）
- **门店贡献（Store contribution）**：对 GMV 变化的贡献拆解（定位主要下滑门店）
- **下单方式占比（如一口价/拼好饭等）**：用于观察订单结构变化（仅作为辅助解释，不作为唯一结论）

## 4. 结果产出（建议从这里开始看）

### Notebook
- `notebooks/01_build_public.ipynb`：构建公开数据（聚合 + 匿名化 + 导出）
- `notebooks/02_insights_report_public.ipynb`：诊断报告（趋势 + 峰值分析 + 下滑贡献 + 解释）

### 典型输出
- GMV 趋势、订单趋势
- 峰值日拆解（Top 门店贡献）
- 门店 GMV 下滑贡献 TopN
- 结构变化与下滑关系的探索性检验（相关/分组对比）
